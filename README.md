# Flag

Flag is a package for the text editor [Atom](https://atom.io/).

It lets you easily create comment flags to visually subdivide your code.

```
**************************************************
*                    My text                     *
**************************************************

##################################################
#                    My text                     #
##################################################

/*************************************************
*                    My text                     *
*************************************************/
```

Supported languages as of today:
- CoffeeScript
- CSS
- HTML
- Javascript.
- LESS
- Python
- SCSS
